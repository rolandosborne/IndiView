# What is IndiView?
IndiView is a social media alternative designed to work with the self-hosted network [diatum](https://diatum.org). With IndiView, you can maintian your contact list, share pictures and videos, and chat all while having full privacy and control of your data. Your data is transmitted only between self-hosted nodes and your device. IndiView maintains a server only to provide a searchable list of public profiles for when you want to add a new contact.
